# dst-mod-keep-following

## Keep Following

Mod for the game [Don't Starve Together][]:

`Shift` + `LMB` on the player or entities below to keep following:

- Chester/Hutch
- Critter
- Glommer
- Pig

Use `WASD` movement keys to stop.
